# 课堂投票系统简介 python1074

## 项目概述

本系统是基于Python的Django框架开发的一款B/S架构的Web系统，旨在提供便捷的课堂投票功能，支持学生与教师之间的互动。

## 技术栈

- 架构：B/S、MVC
- 开发环境：PyCharm、Python 3.x、MySQL
- 技术支持：Python、MySQL、Django、HTML/CSS

## 功能模块

### 系统角色

- 学生
- 管理员（教师）

### 前台系统

- 登录/注册
- 投票参与
- 投票结果查询

### 后台管理系统

#### 管理员端

- **账号管理**
  - 管理注册用户权限（学生、管理员）
  - 支持用户信息修改与删除

- **发布投票**
  - 创建投票主题、设置选项、截止时间及规则
  - 支持实时发布与修改

- **投票管理**
  - 查看历史投票列表
  - 支持按主题搜索、修改投票状态或删除过期投票

- **投票分析**
  - 实时统计投票结果
  - 提供可视化图表（如男女比例、区域分布等）

- **信息管理**
  - 审核学生注册信息，确保用户数据的合法性与完整性

#### 学生端

- **投票参与**
  - 浏览当前开放的投票主题，匿名或实名参与投票

- **我的投票**
  - 查看个人历史投票记录及详细投票内容

- **投票分析**
  - 查看已参与投票的统计结果
  - 支持图表展示与数据导出

- **个人信息**
  - 完善个人资料（性别、生源地等）
  - 支持密码修改与账号安全设置
